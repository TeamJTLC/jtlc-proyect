Referencias pasos

  • Carga de Imagen:

    Carge la imagen del experimento obtenida luego de procesar las muestras.
    Haga clic sobre el recuadro para elejir una imagen o arrastre una imagen desde una carpeta para luego soltarla dentro del área de imagen.
    Para quitar la imagen cargada haga doble clic sobre la misma.

  • Recorte de Imagen:

    Mueva los sliders para seleccionar las partes de la imagen que desea quitar del análisis, resultando del recorte la imagen delimitada por el área interna definida por las líneas de guía.
    La posición inicial de estos sliders es determinada automáticamente por el asistente de proceso.

  • Rotacion de Imagen:

    En este paso puedes rotar la imagen para corregir la alineación de las muestras, asi como espejar la imagen (invertir), realizar una rotación estricta o libre.
    Al rotar la imagen esta se agranda automaticamente (llenando los nuevos espacios con fondo blanco) para evitar perder partes de la misma.
    Utilice las lineas de guía como referencia al momento de alinear las muestras.

  • Separación de Muestras:

    Aquí puede seleccionar las muestras individuales. Utilice los sliders para seleccionar cada muestra, haga doble clic sobre la imagen para agregar una nueva área de selección, seleccione un slider y presione la tecla Suprimir o Retroceso para quitar la selección.
    La posición inicial de las muestras es calculada automáticamente por el asistente de proceso.
    Debe seleccionar al menos una muestra no vacía para poder continuar.

  • Datos de Muestras:

    Ingrese los datos particulares de cada muestra. Utilice los sliders para seleccionar la altura del frente solvente y el punto de siembra. Presione el bóton con el ícono del eslabón o cadena para sincronizar los sliders de todas las muestras. Puede cambiar el nombre de la muestra haciendo clic sobre el bóton con el ícono de un lápiz.
    Presione los botones con íconos de flecha izquierda/derecha para ver las muestras ocultas si se diera el caso en que estas no sean visibles actualmente.

  • Analisis de Muestras:

    Aquí puede seleccionar los picos y el área de integración para cada muestra.
    Utilice los sliders para seleccionar los diferentes picos en la muestra, haga doble clic sobre la imagen o el gráfico para agregar una nueva área de selección, seleccione un slider y presione la tecla Suprimir o Retroceso para quitar la selección.
    Debe seleccionar al menos un pico (no vacío) por cada muestra.
    Las lineas rojas sobre la imagen de la muestra indican el Punto de Siembra y el Frente Solvente previamente establecidos.
    La posición inicial de los sliders/picos es determinada automaticamente por el asistente de proceso.

  • Comparación de Muestras:

    En esta vista puede observar las diferencias en la media de cada muestra. Haciendo clic sobre el ícono de color de la muestra puede elejir un color diferente para cada una de ellas, puede activar o desactivar muestras (curvas) haciendo clic sobre el checkbox.

  • Resultados del Analisis:

    Observe los resultados del analisis para cada muestra. Los resultados de cada pico analizado se observan en el panel ubicado a la derecha, haga clic sobre la etiqueta del pico para ver mas detalles. Despliegue el panel de un pico y presione sobre el boton con el ícono de un lápiz para cambiar el nombre del pico.
