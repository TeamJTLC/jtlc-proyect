Referências etapas

  • Carregar imagem:

    Carregar a imagem Experiência obtida após o processamento das amostras.
    Clique no Box para escolher uma imagem ou arrastar uma imagem de uma pasta e, em seguida, solte-o na a área da imagem.
    Para remover a imagem carregada, clique duas vezes sobre ele.

  • Aparar imagem:

    Mova os controles deslizantes para selecionar partes da imagem a ser removido a partir da análise, resultante da colheita a imagem delimitada pela área interna definido nas orientações.
    A posição inicial destes controles deslizantes é determinado automaticamente pelo processo do assistente.

  • Rotação imagem:

    Nesta etapa, você pode girar a imagem para corrigir a alinhamento das amostras, bem como para espelhar imagem (reverso), execute uma rotação estrita ou livre.
    Ao girar a imagem é automaticamente ampliada (o preenchimento do novo espaço com fundo branco) para evitar a perda de partes do mesmo. Use linhas de guia como uma referência ao alinhar as amostras.

  • Separação amostra:

    Aqui você pode selecionar amostras individuais. Use os controles deslizantes para selecionar cada amostra, faça duplo clique na imagem para Adicionar a nova área de seleção, selecione um controle deslizante e pressione Suprimir ou Backspace remover a seleção.
    A posição inicial das amostras é calculado automaticamente pelo assistente de processo.
    Deve selecionar pelo menos uma amostra não-vazia para continuar.

  • Dados de amostra:

    Digite os detalhes de cada amostra. Use os controles deslizantes para selecionar a altura do contra solvente ponto e semente. Pressione o botão com o link ícone ou eslabon para sincronizar os controles deslizantes de todas as amostras. Você pode alterar o nome da amostra clicando no botão com o ícone de um lápis.
    Pressione os botões com ícones seta esquerda / direita para ver amostras dissimulada, se fosse o caso de que estes não são visíveis hoje.

  • Análise de amostras:

    Aqui você pode selecionar os picos e área de integração para cada amostra. Use os controles deslizantes para selecionar diferentes picos na amostra, faça duplo clique na imagem ou gráfico para Adicionar a nova área de seleção, selecione um controle deslizante e pressione Suprimir ou Backspace para remover a seleção.
    Deve selecionar pelo menos um pico (não vazia) para cada amostra.
    Linhas vermelho sobre a imagem da amostra indicam ponto Semear e Solvente Frente, previamente estabelecido.
    A posição inicial dos controles deslizantes / picos é determinado automaticamente pelo processo de assistente.

  • Comparação da amostra:

    Nesta visualização, você pode ver as diferenças no média de cada amostra. Clicando a cor do ícone da amostra pode escolher um diferente para cada cor, você pode ativar ou desativar amostras (curvas), clicando na caixa de seleção.

  • Resultados da análise:

    Olhe para os resultados da análise para cada amostra. Analisaram os resultados de cada pico é observado no painel à direita, clique no rótulo do pico para ver mais detalhes. Puxe para baixo o painel de um pico e clique no botão com o ícone de um lápis para mudar o nome do pico.
